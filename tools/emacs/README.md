


## 安装

```bash
sudo apt install emacs
```


三种模式:

| 模式     | 说明                                     |
|----------|------------------------------------------|
| Terminal | 终端模式                                 |
| Gui      | 图形界面模式                             |
| Client   | 客户端模式，类似图形界面(配合daemon使用) |



## mode-line 说明

```bash
U:**-
```

### 字符编码

| 字符 | 说明                  |
|------|-----------------------|
| U    | UTF-8 编码            |
| 1    | ISO-8859-1            |
| -    | 未识别或者默认(ASCII) |
| =    | ？？见到再说          |

### 保存状态

| 字符 | 说明             |
|------|------------------|
| --   | 没被修改过       |
| **   | 已被修改过       |
| %%   | 文件只读         |
| %*   | 只读但是做了修改 |

### 换行符格式

| 字符 | 说明                |
|------|---------------------|
| -    | Unix 格式，LF 换行  |
| :    | Dos 格式，CRLF 换行 |
| /    | Mac 格式，CR 换行   |
