

## 帮助

| 快捷键 | 说明                            |
|--------|---------------------------------|
| C-h k  | 查询快捷键信息                  |
| C-h f  | 查询函数信息                    |
| C-h v  | 查询变量帮助信息                |
| C-h m  | 查询当前模式信息                |
| C-h b  | 查询当前buffer 可用的全部快捷键 |


## 退出

| 快捷键  | 说明                  |
|---------|-----------------------|
| C-x C-c | 程序退出              |
| C-z     | 临时退出Emacs(fg恢复) |


## 文件

| 快捷键  | 说明                       |
|---------|----------------------------|
| C-x C-f | 打开文件(可以用于打开目录) |
| C-x C-w | 文件另存为                 |
| C-x C-s | 保存文件                   |


## 移动

| 快捷键 | 单位   | 说明                   |
|--------|--------|------------------------|
| C-p    | 行     | 向上一行               |
| C-n    | 行     | 向下一行               |
| C-b    | 字符   | 向右移动一个字符       |
| C-f    | 字符   | 向左移动一个字符       |
| M-f    | 词     | 向前移动一个词         |
| M-b    | 词     | 向后移动一个词         |
| C-a    | 行内   | 移动到行首             |
| C-e    | 行内   | 移动到行尾             |
| M-a    | 句     | 移动到句首             |
| M-e    | 句     | 移动到句尾             |
| M-{    | 段     | 向上一段               |
| M-}    | 段     | 向下一段               |
| C-v    | page   | 向下翻页; 向下滚动窗口 |
| M-v    | page   | 向上翻页; 向上滚动窗口 |
| M-<    | buffer | buffer头               |
| M->    | buffer | buffer尾               |


## 查找

| 快捷键 | 说明     |
|--------|----------|
| C-s    | 向前查找 |
| C-r    | 向后查找 |


## 编辑

| 快捷键  | 说明               |
|---------|--------------------|
| <del>   | 删除前面一个字符   |
| C-d     | 删除后面一个字符   |
| M-<del> | 删除前面一个词     |
| M-d     | 删除后面一个词     |
| C-k     | 删除光标位置到行末 |
| M-k     | 删除光标位置到句末 |
| C-w     | 删除选中位置       |
| C-y     | 粘贴               |
| C+/     | 撤销               |


## 窗口

| 快捷键 | 说明                         |
|--------|------------------------------|
| C-x 2  | 水平分割，上下叠放           |
| C-x 3  | 垂直分割，左右并排           |
| C-x 0  | 关闭当前窗口                 |
| C-x 1  | 只保留当前窗口，关闭其他窗口 |
| C-x o  | 窗口切换                     |
| C-x O  | 窗口切换(反方向)             |
| C-M-v  | 滚动另一个窗口               |


## Frames

| 快捷键  | 说明               |
|---------|--------------------|
| C-x 5 2 | 开启一个新的 frame |
| C-x 5 0 | 删除选中的 frame   |


## 缓冲区

| 快捷键  | 说明             |
|---------|------------------|
| C-x k   | 关闭当前缓冲区   |
| C-x C-b | 查看所有缓冲区   |
| C-x b   | 跳转到指定缓冲区 |
| C-x s   | 保存指定缓冲区   |


## 其他功能

| 快捷键 | 说明                       |
|--------|----------------------------|
| C-u    | 重复执行次数               |
| C-g    | 停止命令执行; 取消执行次数 |
| C-l    | 滚动窗口使光标居中         |
| C-h c  | 显示帮助信息               |
| C-h k  | 快捷键查询                 |
| C-h r  | 查看manual                 |
