
## 帮助

| 快捷键 | 命令                 | 说明                            |
|--------|----------------------|---------------------------------|
| C-h k  | describe-key         | 查询快捷键信息                  |
| C-h f  | describe-function    | 查询函数信息                    |
| C-h v  | describe-variable    | 查询变量帮助信息                |
| C-h m  | describe-mode        | 查询当前模式信息                |
| C-h b  | describe-bindings    | 查询当前buffer 可用的全部快捷键 |


## 退出

| 快捷键  | 命令                       | 说明                  |
|---------|----------------------------|-----------------------|
| C-x C-c | save-buffers-kill-terminal | 程序退出              |
| C-z     | suspend-frame              | 临时退出Emacs(fg恢复) |


## 文件

| 快捷键  | 命令        | 说明                       |
|---------|-------------|----------------------------|
| C-x C-f | find-file   | 打开文件(可以用于打开目录) |
| C-x C-w | write-file  | 文件另存为                 |
| C-x C-s | save-buffer | 保存文件                   |


## 移动

| 快捷键 | 命令                   | 单位   | 说明                   |
|--------|------------------------|--------|------------------------|
| C-p    | previous-line          | 行     | 向上一行               |
| C-n    | next-line              | 行     | 向下一行               |
| C-b    | backward-char          | 字符   | 向右移动一个字符       |
| C-f    | forward-char           | 字符   | 向左移动一个字符       |
| M-f    | forward-word           | 词     | 向前移动一个词         |
| M-b    | backward-word          | 词     | 向后移动一个词         |
| C-a    | move-beginning-of-line | 行内   | 移动到行首             |
| C-e    | move-end-of-line       | 行内   | 移动到行尾             |
| M-a    | backward-sentence      | 句     | 移动到句首             |
| M-e    | forward-sentence       | 句     | 移动到句尾             |
| M-{    | backward-paragraph     | 段     | 向上一段               |
| M-}    | forward-paragraph      | 段     | 向下一段               |
| C-v    | scroll-up-command      | page   | 向下翻页; 向下滚动窗口 |
| M-v    | scroll-down-command    | page   | 向上翻页; 向上滚动窗口 |
| M-<    | beginning-of-buffer    | buffer | buffer头               |
| M->    | end-of-buffer          | buffer | buffer尾               |


## 查找

| 快捷键 | 命令             | 说明     |
|--------|------------------|----------|
| C-s    | isearch-forward  | 向前查找 |
| C-r    | isearch-backward | 向后查找 |


## 编辑

| 快捷键 | 命令                         | 说明               |
|--------|------------------------------|--------------------|
| DEL    | backward-delete-char-untabify | 删除前面一个字符   |
| C-d    | delete-char                  | 删除后面一个字符   |
| M-DEL  | backward-kill-word           | 删除前面一个词     |
| M-d    | kill-word                    | 删除后面一个词     |
| C-k    | kill-line                    | 删除光标位置到行末 |
| M-k    | kill-sentence                | 删除光标位置到句末 |
| C-w    | kill-region                  | 删除选中位置       |
| C-y    | yank                         | 粘贴               |
| M+/    | dabbrev-expand               | 自动补全           |
| C+/    | undo                         | 撤销               |


## 窗口

| 快捷键 | 命令                 | 说明                         |
|--------|----------------------|------------------------------|
| C-x 2  | split-window-below   | 水平分割，上下叠放           |
| C-x 3  | split-window-right   | 垂直分割，左右并排           |
| C-x 0  | delete-window        | 关闭当前窗口                 |
| C-x 1  | delete-other-windows | 只保留当前窗口，关闭其他窗口 |
| C-x o  | other-window         | 窗口切换                     |
| C-x O  | 无默认绑定           | 窗口切换(反方向)             |
| C-M-v  | scroll-other-window  | 滚动另一个窗口               |


## Frames

| 快捷键  | 命令               | 说明               |
|---------|--------------------|--------------------|
| C-x 5 2 | make-frame-command | 开启一个新的 frame |
| C-x 5 0 | delete-frame       | 删除选中的 frame   |


## 缓冲区

| 快捷键  | 命令              | 说明             |
|---------|-------------------|------------------|
| C-x k   | kill-buffer       | 关闭当前缓冲区   |
| C-x C-b | list-buffers      | 查看所有缓冲区   |
| C-x b   | switch-to-buffer  | 跳转到指定缓冲区 |
| C-x s   | save-some-buffers | 保存指定缓冲区   |


## 其他功能

|--------|----------------------|----------------------------|
| C-u    | universal-argument   | 重复执行次数               |
| C-g    | keyboard-quit        | 停止命令执行; 取消执行次数 |
| C-l    | recenter-top-bottom  | 滚动窗口使光标居中         |
| C-h c  | describe-key-briefly | 显示帮助信息               |
| C-h k  | describe-key         | 快捷键查询                 |
| C-h r  | info-emacs-manual    | 查看manual                 |
