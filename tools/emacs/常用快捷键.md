
## 退出

| 快捷键  | 说明     |
|---------|----------|
| C-x C-c | 程序退出 |


## 文件

| 快捷键  | 说明                       |
|---------|----------------------------|
| C-x C-f | 打开文件(可以用于打开目录) |
| C-x C-w | 文件另存为                 |
| C-x C-s | 保存文件                   |


## 移动

| 快捷键 | 单位   | 说明                   |
|--------|--------|------------------------|
| C-p    | 行     | 向上一行               |
| C-n    | 行     | 向下一行               |
| C-b    | 字符   | 向右移动一个字符       |
| C-f    | 字符   | 向左移动一个字符       |
| M-f    | 词     | 向前移动一个词         |
| M-b    | 词     | 向后移动一个词         |
| C-a    | 行内   | 移动到行首             |
| C-e    | 行内   | 移动到行尾             |
| M-a    | 句     | 移动到句首             |
| M-e    | 句     | 移动到句尾             |
| M-{    | 段     | 向上一段               |
| M-}    | 段     | 向下一段               |
| C-v    | page   | 向下翻页; 向下滚动窗口 |
| M-v    | page   | 向上翻页; 向上滚动窗口 |
| M-<    | buffer | buffer头               |
| M->    | buffer | buffer尾               |


## 编辑

| 快捷键 | 单位 | 说明           |
|--------|------|----------------|
| C-d    | 字符 | 删除后一个字符 |


## 窗口

| 快捷键 | 说明                         |
|--------|------------------------------|
| C-x 1  | 只保留当前窗口，关闭其他窗口 |


## 缓冲区

| 快捷键 | 说明           |
|--------|----------------|
| C-x k  | 关闭当前缓冲区 |


## 其他功能

| 快捷键 | 说明                       |
|--------|----------------------------|
| C-u    | 重复执行次数               |
| C-g    | 停止命令执行; 取消执行次数 |
| C-l    | 重新绘制使光标居中         |
