## 移动

| 快捷键 | 单位   | 说明             |
|--------|--------|------------------|
| C-f    | 字符   | 向前移动一个字符 |
| C-b    | 字符   | 向后移动一个字符 |
| M-f    | 词     | 向前移动一个词   |
| M-b    | 词     | 向后移动一个词   |
| C-a    | 行内   | 移动到行首       |
| C-e    | 行内   | 移动到行尾       |
| C-p    | 行     | 向上一行         |
| C-n    | 行     | 向下一行         |
| M-{    | 段     | 向上一段         |
| M-}    | 段     | 向下一段         |
| M-<    | buffer | buffer头         |
| M->    | buffer | buffer尾         |


## 编辑

| 快捷键 | 单位 | 说明           |
|--------|------|----------------|
| C-d    | 字符 | 删除后一个字符 |
