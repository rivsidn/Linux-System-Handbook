
## 表格

### 构建表格

1. 输入Header

   ```org
   | name | phone | Age |
   ```

2. 执行`C-c RET`

  ```org
  | Name | Phone | Age |
  |------+-------+-----|
  |      |       |     |
  ```

### 快捷键

| 快捷键  | 说明                       |
|---------|----------------------------|
| C-c  \| | 创建表格                   |
| C-c C-c | 对齐表格                   |
| Tab     | 对齐表格，移动到下一个区域 |
| S-Tab   | 对齐表格，移动到上一个区域 |
| RET     | 对齐表格，移动到下一行     |
