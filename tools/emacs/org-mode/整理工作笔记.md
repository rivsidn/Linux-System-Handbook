
当前工作笔记是多个工作目录的简单罗列，缺少对当前工作进度的有效跟踪.

org-mode 支持通过TODO 跟踪项目状态，可以尝试一下.

## 文件格式

```org
#+TITLE: 工作状态

#+STARTUP: content
#+TODO: TODO(t) DOING(g!) WAIT(w) | DONE(d!) CANCEL(c!)

#+NAME: task-manager-functions
#+BEGIN_SRC emacs-lisp :results silent
  (defun local/new-workspace-item ()
    (interactive)
    (let* ((title (read-string "工作标题: "))
           (date (format-time-string "%Y-%m-%d"))
           (name (concat date "-" title))
           (dir (expand-file-name name (file-name-directory buffer-file-name)))
           (entry (expand-file-name (concat title ".md") dir)))
      (make-directory dir t)
      (goto-char (point-max))
      (re-search-backward "^# Local Variables:" nil t)
      (beginning-of-line)
      (insert (format "\n** TODO %s\n\n- 目录：[[file:%s/]]\n- 文档：[[file:%s/%s.md]]\n\n\n"
                      name name name title))
      (save-buffer)
      (find-file entry)))
#+END_SRC



* 工作列表

** DONE 2026-05-12-测试

- State "DONE"       from "TODO"       [2026-05-12 Tue 15:34]
- 目录：[[file:2026-05-12-测试/]]
- 文档：[[file:2026-05-12-测试/测试.md]]


# Local Variables:
# eval: (save-excursion (goto-char (point-min)) (org-babel-goto-named-src-block "task-manager-functions") (let ((org-confirm-babel-evaluate nil)) (org-babel-execute-src-block)))
# End:

```

### 快捷键

| 快捷键  | 说明               |
|---------|--------------------|
| C-c C-o | 打开目录链接       |
| C-c C-t | 修改TODO状态       |
| TAB     | 展开/折叠标题      |
| S-TAB   | 切换文件的显示状态 |
| C-c C-c | 执行代码           |

刚写好的TODO、emacs functions 不能立即生效，需要执行`C-c C-c` .

## 相关技术

### 关键字

| 关键字  | 说明                                                       |
|---------|------------------------------------------------------------|
| TITLE   | 标题                                                       |
| STARTUP | 打开文件时显示的方式                                       |
| TODO    | TODO 状态列表，支持用中文，标签后的'!'表示切换时插入时间戳 |

### STARTUP 关键字

| 参数     | 说明                                 |
|----------|--------------------------------------|
| overview | 文件打开时，只显示标题结构，内容折叠 |
| content  | 显示所有标题，正文内容折叠           |
| showall  | 全部展开                             |
