### 列编辑

Emacs 的列编辑通常叫矩形编辑，适合同时操作多行同一列内容，比如批量加前缀、删除一列、对齐文本。

| 快捷键 | 说明 |
|--------|------|
| `C-x SPC` | 进入矩形标记模式 |
| `C-x r t` | 在矩形区域每一行前插入同样内容 |
| `C-x r k` | 删除矩形区域 |
| `C-x r y` | 粘贴矩形区域 |
| `C-x r o` | 打开矩形区域，在每行留出空白 |
| `C-x r c` | 用同一个字符填充矩形区域 |
| `C-g` | 退出矩形标记 |

常见用法：

1. `C-x SPC` 进入矩形选择。
2. 用方向键或 `C-n` `C-p` `C-f` `C-b` 选中多行同一列。
3. 执行矩形命令，比如 `C-x r t` 批量插入，或 `C-x r k` 批量删除。

示例：给多行开头同时加 `# ` 注释。

1. 把光标放到第一行开头。
2. `C-x SPC`。
3. 向下选择多行。
4. `C-x r t`，输入 `# `，回车。

如果只是想像现代编辑器那样直接多光标式列输入，也可以启用 `cua-mode` 后使用矩形编辑，但默认最稳妥的还是 `C-x SPC` 这套内建命令。


### 窗口之间切换

Emacs 里的窗口是同一个界面中的分栏区域，不是操作系统层面的独立程序窗口。日常最常用的是在多个分屏之间切换和关闭窗口。

| 快捷键 | 说明 |
|--------|------|
| `C-x o` | 切换到下一个窗口 |
| `C-x O` | 切换到上一个窗口 |
| `C-x 0` | 关闭当前窗口 |
| `C-x 1` | 只保留当前窗口 |
| `C-x 2` | 上下分割窗口 |
| `C-x 3` | 左右分割窗口 |
| `C-M-v` | 滚动另一个窗口 |

常见用法：

1. `C-x 3` 左右分屏，或 `C-x 2` 上下分屏。
2. `C-x o` 在不同窗口之间切换。
3. `C-x 0` 关闭当前窗口，或 `C-x 1` 只保留当前窗口。

如果你希望按方向切换窗口，也可以使用 `windmove`：

| 命令 | 说明 |
|------|------|
| `M-x windmove-left` | 切换到左边窗口 |
| `M-x windmove-right` | 切换到右边窗口 |
| `M-x windmove-up` | 切换到上边窗口 |
| `M-x windmove-down` | 切换到下边窗口 |

如果后续你觉得 `C-x o` 不够直观，也可以在配置里把 `windmove` 绑定到 `Shift + 方向键`。
