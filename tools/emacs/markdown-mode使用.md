
## 启用模式

| 操作                | 说明                                 |
|---------------------|--------------------------------------|
| `M-x markdown-mode` | 当前 buffer 切换到 `markdown-mode`   |
| `M-x gfm-mode`      | 切换到 GitHub Flavored Markdown 模式 |


## 文档结构

| 快捷键                | 说明                    |
|-----------------------|-------------------------|
| `TAB`                 | 折叠或展开当前标题      |
| `S-TAB` / `<backtab>` | 全文按层级循环折叠/展开 |
| `C-c C-n`             | 跳到下一个标题或列表项  |
| `C-c C-p`             | 跳到上一个标题或列表项  |
| `C-c C-f`             | 跳到下一个同级标题      |
| `C-c C-b`             | 跳到上一个同级标题      |
| `C-c C-u`             | 跳到上一级标题          |
| `C-c C-M-h`           | 选中当前子树            |
| `C-x n s`             | 只查看当前子树          |
| `M-x imenu`           | 按标题索引跳转          |


## 标题与样式

先按 `C-c C-s`，再按下表中的第二个按键：

| 按键                    | 说明                    |
|-------------------------|-------------------------|
| `h`                     | 智能插入标题            |
| `1` `2` `3` `4` `5` `6` | 插入 1 到 6 级 ATX 标题 |
| `b`                     | 加粗                    |
| `i`                     | 斜体                    |
| `s`                     | 删除线                  |
| `c`                     | 行内代码                |
| `C`                     | GFM 代码块              |
| `l`                     | 插入链接                |
| `q`                     | 插入引用块              |
| `t`                     | 插入表格                |
| `[`                     | 插入 GFM checkbox       |
| `-`                     | 插入分隔线              |


## 链接与跳转

| 快捷键    | 说明           |
|-----------|----------------|
| `C-c C-l` | 插入链接       |
| `C-c C-i` | 插入图片       |
| `C-c C-o` | 打开光标处链接 |
| `M-n`     | 跳到下一个链接 |
| `M-p`     | 跳到上一个链接 |

目录文档中的常见工作流：

1. 在 Markdown 目录文件中，把光标放到某个链接上。
2. 按 `C-c C-o` 打开对应的本地 Markdown 文件。
3. 编辑完成后，按 `C-x LEFT` 返回刚才的目录文档。
4. 如果想再次前进回来，可以按 `C-x RIGHT`。


## 列表编辑

| 快捷键        | 说明                         |
|---------------|------------------------------|
| `M-RET`       | 新建同级列表项               |
| `C-c C-j`     | 新建列表项                   |
| `RET`         | 在列表中继续下一项           |
| `DEL`         | 需要时减少缩进，否则正常删除 |
| `C-c <left>`  | 提升标题或列表层级           |
| `C-c <right>` | 降低标题或列表层级           |
| `C-c <up>`    | 上移当前标题或列表项         |
| `C-c <down>`  | 下移当前标题或列表项         |
| `C-c C-c n`   | 重新整理有序列表编号         |


## 表格

| 快捷键          | 说明             |
|-----------------|------------------|
| `C-c C-s t`     | 插入表格         |
| `C-c S-<right>` | 插入列           |
| `C-c S-<left>`  | 删除列           |
| `C-c S-<down>`  | 插入行           |
| `C-c S-<up>`    | 删除行           |
| `C-c C-c t`     | 转置表格         |
| `C-c C-c ^`     | 按列排序         |
| `C-c C-c |`     | 将区域转换成表格 |


## 代码块与预览

| 快捷键      | 说明              |
|-------------|-------------------|
| `C-c C-s C` | 插入 GFM 代码块   |
| `C-c '`     | 编辑当前代码块    |
| `C-c C-c p` | 预览 Markdown     |
| `C-c C-c e` | 导出 Markdown     |
| `C-c C-c v` | 导出并预览        |
| `C-c C-c o` | 打开导出后的文件  |
| `C-c C-c l` | 切换 live preview |


## 显示切换

| 快捷键        | 说明                         |
|---------------|------------------------------|
| `C-c C-x C-m` | 隐藏或显示 Markdown 标记符号 |
| `C-c C-x C-i` | 隐藏或显示行内图片           |
| `C-c C-x C-l` | 隐藏或显示链接 URL           |
| `C-c C-x C-f` | 切换代码块原生高亮           |


## 当前环境附加快捷键

这些不是 `markdown-mode` 默认绑定，是当前 `init.el` 里的自定义：

| 快捷键 | 说明                           |
|--------|--------------------------------|
| `F8`   | 打开并聚焦 `imenu-list` 侧边栏 |
| `F9`   | 在右侧打开当前目录的 `dired`   |


## 说明

- `C-c C-s` 是最常用的样式前缀键，标题、加粗、代码块、表格都从这里进入。
- `C-c C-c` 是命令前缀键，预览、导出、整理编号、表格处理都从这里进入。
- 如果忘了按键，可以用 `C-h k` 先按快捷键查询，或者 `M-x describe-mode` 查看当前模式帮助。
