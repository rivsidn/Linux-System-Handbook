## 安装

进入到`.vim/bundle/` 下执行命令：

```bash
git clone https://github.com/vim-scripts/DrawIt
```



## 使用

| 动作  | 命令说明                              |
| ----- | ------------------------------------- |
| \di   | 开启 DrawIt，开启之后默认进入画图模式 |
| \ds   | 关闭 DrawIt                           |
| space | 画图/擦除模式切换                     |
| Up    | 上直线                                |
| Down  | 下直线                                |
| Left  | 左直线                                |
| Right | 右直线                                |
| Home  | 左上                                  |
| End   | 左下                                  |
| Pgup  | 右上                                  |
| Pgdn  | 右下                                  |
| ^     | 上箭头                                |
| \^    | 上箭头（粗）                          |
| v     | 下箭头                                |
| \v    | 下箭头（粗）                          |
| >     | 右箭头                                |
| \\>   | 右箭头（粗）                          |
| <     | 左箭头                                |
| \\>   | 左箭头（粗）                          |
|       |                                       |
| \b    | 矩形                                  |
| \e    | 椭圆                                  |
| \l    | 折线                                  |
| \a    | 折线（带箭头）                        |
| \f    | 填充                                  |
|       |                                       |
