## 使用命令

| 命令  | 说明                 |
| ----- | -------------------- |
| mm    | 无名书签             |
| mi    | 添加有注释的书签     |
| mn/mp | 后一个/前一个书签    |
| ma    | 显示所有书签         |
| mc    | 删除当前buffer中书签 |
| mx    | 删除所有书签         |
