

### vimgrep

| 命令                             | 说明                                          |
| -------------------------------- | --------------------------------------------- |
| :vimgrep /\<pattern\>/ Makefile\* | 文件搜索                                      |
| :copen                           | 搜索之后打开(Quickfix Window)，显示所有匹配项 |
| :cn                              | 下一个匹配项                                  |
| :cp                              | 上一个匹配项                                  |
